## Skill Installation Guidelines

### 1. Search Phase

- Search for available skills matching the requirements
- Check skill repositories and registries
- Review skill descriptions and compatibility
- Identify the most suitable skill

### 2. Review Phase

Before installation, review:

- Skill documentation and README
- Required dependencies and prerequisites
- Compatibility with current workspace
- Security considerations
- License and usage terms

### 3. Installation Phase

Install the skill by:

- Downloading skill files to `.opencodex/skills/[skill-name]/`
- Installing any required dependencies
- Verifying file integrity
- Setting appropriate permissions

### 4. Configuration Phase

Configure the skill:

- Update workspace settings if needed
- Set environment variables
- Configure skill-specific parameters
- Test basic functionality

### 5. Verification Phase

Verify successful installation:

- Check skill is discoverable
- Test skill with example inputs
- Verify all dependencies are met
- Confirm integration with existing tools

## Output Requirements

Provide clear feedback including:

- Skill name and version installed
- Installation location
- Configuration steps taken
- Verification results
- Next steps for using the skill
- Any warnings or issues encountered

If installation fails, explain:

- What went wrong
- Possible causes
- Suggested solutions
- Alternative skills to consider
